# 更新日志 CHANGELOG

## 2026-04-10
- `build_prompt` 强化：【排版指令】要求大量恰当使用微信风格 Emoji；【字段指令】活动标题以震撼符号开头、卖点每条以 ✅ 开头

## 2026-04-09
- 调整 `normalize_output` 的收口策略：卖点、详细描述、标签数量不足时不再直接返回 `None`，改为降级保留已有内容
- 标题字段新增轻量兜底：缺失活动标题或商品标题时，优先使用另一字段回填，避免整包生成失败
- 调整 `build_prompt`：新增标题 Emoji、卖点前缀符号、详细描述段落呼吸感等排版规范约束
- 优化价格提示逻辑：同时有团购价和原价时，要求文案突出差价与划算感表达
- 强化详细描述结构要求：明确采用“痛点/场景 + 解决方案 + 下单理由”的带货逻辑，并加入示例句式
- 强化 `prompt.py` 的系统角色设定：改为“拥有 5 年经验、10 万+ 跟团次数”的快团团食品饮料资深运营专家
- 新增文案语气约束：要求输出更亲切、真诚、富有感染力，禁止公文式、说明书式和生硬参数堆砌表达
- 同步更新 `CONTEXT.md`，记录 Prompt 专家人设与高转化语言风格已完成
